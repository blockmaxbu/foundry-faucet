## Installation
1. Clone this repo:
   ```bash
   git clone https://github.com/blockmaxbu/foundry-faucet.git
   ```
2. Install dependencies:
   ```bash
   make install
   ```
3. deply the project locally:
   ```bash
   make deploy
   ```
